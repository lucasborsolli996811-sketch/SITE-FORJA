🌐 PUBLICAR NA INTERNET

Quer que seu site fique acessível de qualquer lugar?

OPÇÃO 1: NETLIFY (Recomendado - Grátis)
  1. Vá para netlify.com
  2. Faça login (ou crie conta)
  3. Arraste a pasta SITE inteira
  4. Pronto! Seu site está online!

OPÇÃO 2: GITHUB PAGES (Grátis)
  1. Crie conta em github.com
  2. Crie novo repositório
  3. Upload da pasta SITE
  4. Ative GitHub Pages
  5. Seu site estará em username.github.io/SITE

OPÇÃO 3: SEU PRÓPRIO DOMÍNIO
  1. Registre domínio (ex: meudominio.com.br)
  2. Hospede em qualquer servidor (Hostinger, Bluehost)
  3. Faça upload via FTP

❓ DÚVIDAS FREQUENTES

P: Preciso de HTTPS?
R: Sim, para publicar. Mas localmente (http://) funciona OK.

P: Posso usar sem internet?
R: Sim! Apenas clique no SITE.html (sem servidor).
  Haverá aviso, mas funciona na maioria dos navegadores.

P: Como faço backup?
R: Copie a pasta SITE inteira para outro lugar.

P: Posso usar meu próprio domínio?
R: Sim! Registre domínio e aponte para seu servidor.

P: Preciso de banco de dados?
R: Não por enquanto. Para formulários avançados, sim.

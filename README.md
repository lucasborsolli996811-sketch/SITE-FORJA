# 🔧 FORJA | IDE v1.0.0

Uma interface visual profissional inspirada no VS Code para apresentar a empresa **FORJA** - especializada em Engenharia, Desenho Técnico e Impressão 3D.

## 📋 Conteúdo do Projeto

```
SITE/
├── SITE.html          # Arquivo principal HTML
├── style.css          # Estilos CSS (tema escuro VS Code)
├── script.js          # JavaScript para interatividade
├── README.md          # Este arquivo
└── Imagem/            # Pasta para imagens (opcional)
```

## ✨ Características

- ✅ **Interface VS Code**: Replica o design visual do Visual Studio Code
- ✅ **Navegação Suave**: Transição suave entre seções
- ✅ **Tema Escuro Profissional**: Cores otimizadas para leitura
- ✅ **Responsivo**: Adapta-se a diferentes tamanhos de tela
- ✅ **Interatividade**: Cliques e efeitos visuais
- ✅ **Font Awesome Icons**: Ícones profissionais integrados
- ✅ **Acessibilidade**: Estrutura semântica HTML5

## 🎯 Funcionalidades JavaScript

- **Navegação com Âncoras**: Clique em um arquivo para ir para sua seção
- **Scroll Suave**: Transição suave ao navegar entre seções
- **Ícones Interativos**: Efeitos hover nos ícones
- **Status Bar Dinâmica**: Barra de status visual
- **Atalhos Teclado**: Ctrl+P para "Quick Open" (extensível)

## 🛠️ Customização

### Mudar Cores
Edit `style.css` - Variáveis CSS no topo:
```css
:root {
    --bg-primary: #1e1e1e;
    --accent-blue: #007acc;
    /* ... mais cores ... */
}
```

### Adicionar Novos Arquivos
1. Adicione `<li>` na lista de arquivos em `SITE.html`
2. Crie uma nova `<section>` com `id` correspondente
3. Adicione estilos CSS se necessário

### Adicionar Ícones
Todos os ícones usam **Font Awesome 6.0**. Confira: https://fontawesome.com/icons
